# The Condenser

Mobile-first punch list management for residential construction managers.

## What It Does

- Create and manage punch lists per lot/address
- Smart trade classification and auto-assignment
- Photo capture and annotation
- One-tap email/text generation for trade partners
- Offline-first — works in the field with no signal
- Installable as a PWA on any phone

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Dexie (IndexedDB) for local-first storage
- Vite + PWA plugin
- Express backend with Prisma

## Getting Started

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm start
```
